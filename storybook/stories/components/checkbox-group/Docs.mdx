# Checkbox Group

Checkbox group component is a view container of multiple checkboxes with group label and optionally with helper text. You can see the [anatomy](https://carbondesignsystem.com/components/checkbox/usage/#anatomy)

```tsx
import {
  CheckboxGroup,
} from '@audira/carbon-react-native'

export default function Component() {

  return (
    <CheckboxGroup legend="Legend Text">
      <CheckboxGroup.Item
        label="Item 1"
      />
      <CheckboxGroup.Item
        label="Item 2"
      />
      <CheckboxGroup.Item
        label="Item 3"
      />
    </CheckboxGroup>
  )

}
```

Do not be confused of that `<CheckboxGroup.Item>`. It is same as the [`<Checkbox>`](?path=/story/components-checkbox--checkbox)

> Currently, it doesn't support of [nesting](https://carbondesignsystem.com/components/checkbox/usage/#nesting) the checkbox item

## Stack Orientation

Checkboxes in a group will be stacked vertically by default, but you can make it stacked horizontally. Pass `horizontal` to the `orientation` prop

## Helper Text
