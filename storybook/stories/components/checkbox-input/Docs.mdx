# Checkbox Input

This component provides only the box input. Mostly, you only need [Checkbox](?path=/docs/components-checkbox--docs) or [CheckboxGroup](?path=/docs/components-checkbox-group--docs). This component might suit your needs if you only want to render the box

```tsx
import {
  CheckboxInput,
} from '@audira/carbon-react-native'

export default function Component() {
  return (
    <CheckboxInput
      defaultChecked // make it ticked initially
    />
  )
}
```
