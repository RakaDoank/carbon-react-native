# Radio Button Input

This component provides only the circle input. Mostly, you only need [RadioButtonGroup](?path=/docs/components-radio-button-group--docs). This component might suit your needs if you only want to render the circle.

```tsx
import {
  RadioButtonInput,
} from '@rakadoank/carbon-react-native'

export default function Component() {
  return (
    <RadioButtonInput value="bbc"/>
  )
}
```

## Caveat

In common usage, radio button usually represent a group of mutually exclusive choices, compared to checkboxes that user is able to select one or more.

Probably you only need [RadioButtonGroup](?path=/docs/components-radio-button-group--docs) instead of manually building group of radio buttons.
